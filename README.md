# Android LightWeight Squicle Library for compose


# Installation
- Add jitpack.io to your project
```
repositories {
    ...
    maven { url "https://jitpack.io" }
}
```
- Add squircle library dependency
``` 
dependencies { 
  ...
  implementation 'com.github.Size0f:android.compose.squircle:1.0.5'
}
```
